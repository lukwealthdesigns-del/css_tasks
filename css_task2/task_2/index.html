This layout uses CSS Grid template areas to create a classic website structure. The grid areas are defined as:

  • header header header - spans all 3 columns
  • sidebar main main - sidebar takes 1 column, main takes 2
  • footer footer footer - spans all 3 columns

This approach makes layouts incredibly intuitive and easy to modify!
